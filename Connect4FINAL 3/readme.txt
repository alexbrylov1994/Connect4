---==== CONNECT-4 LOADING INSTRUCTIONS FOR PYTHON 3.3.2 ====----


For console based interaction:

1)Extract files from Connect4Final.zip to a separate folder.
2)Open the Terminal Emulator(on Linux computers)/Command Prompt(on Windows computers)/Terminal(on MacOS computers)
3)Change the directory to the <folder_name>/Connect4
4)Run main.py using the python interpreter to start the game in the console 
5)Enjoy!

For GUI/Window based interaction:

1)Extract files from Connect4Final.zip to a separate folder.
2)Open the Terminal Emulator(on Linux computers)/Command Prompt(on Windows computers)/Terminal(on MacOS computers)
3)change the directory to the <folder_name>/Connect4
4)Run gui.py using the python interpreter to start the game. A separate window with the game will be opened.
5)Enjoy!

---==== CONNECT-4 FILE LIST ====----

1)ai_player.py - Responsible for ai interaction and ai move selection
2)grid.py - Responsible for board management
3)gui.py - Responsible for presenting the game in a visually appropriate way using GUI
4)main.py - Responsible for running the game in the console in a visually appropriate way
5)player.py - Responsible for user interaction and user input handling(to prevent the game from crashing based on user input provided)
